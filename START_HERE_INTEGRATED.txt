5. How to start the full project

Step A - Start the Python ML service
Command:
cd backend-spring\attijari-compass-ml
python -m pip install -r requirements.txt
python train.py
python -m uvicorn app:app --host 127.0.0.1 --port 8001 --reload

Quick check:
http://127.0.0.1:8001/health


Step B - Start the Spring Boot backend
Recommended from the backend folder:
cd backend-spring
.\mvnw.cmd spring-boot:run

If the Maven wrapper does not start correctly on your terminal:
- run it from IntelliJ
- or install Maven and use:
  mvn spring-boot:run

Backend URL:
http://localhost:8082


Step C - Start the Angular frontend
Command:
cd frontend-angular
npm install
npm start

Frontend URL:
http://localhost:4200
