# Storytelling Assistant Assets

Placez ici les 2 images statiques du personnage :

- `assistant-eyes-open.png`
- `assistant-eyes-closed.png`

Recommandations :

- format PNG ou JPG
- ratio proche de 4:5
- resolution conseillee : 900x1125 px ou plus
- meme cadrage pour les deux images afin que le clignement reste naturel

Si ces fichiers ne sont pas encore presents, le frontend utilise temporairement `src/assets/agent-eyes-open.jpg` comme image de secours.
